# Government Hospitals and Beds Analysis (Power BI)

**Project Overview**

This project analyzes the distribution of government hospitals and hospital beds across different states and regions in India. The analysis is presented through an interactive Power BI dashboard that helps visualize healthcare infrastructure data in a clear and meaningful way.

The dashboard provides insights into the availability of hospitals and beds across states, helping users understand how healthcare resources are distributed geographically.



## Background

Healthcare infrastructure plays an important role in ensuring the well-being of a country’s population. Government hospitals provide affordable and accessible healthcare services to millions of people across India.

However, the number of hospitals and available hospital beds can vary significantly from one state to another. Understanding these variations helps highlight areas where healthcare resources may need improvement.

This project was developed to analyze government hospital infrastructure using data visualization. By building an interactive dashboard in Power BI, the project enables users to explore state-wise and region-wise healthcare data and better understand the distribution of healthcare facilities.



## Objectives

The main objectives of this project are:

* To analyze the distribution of government hospitals across Indian states.
* To examine the availability of hospital beds in different regions.
* To visualize healthcare infrastructure data using an interactive dashboard.
* To help identify patterns and differences in healthcare resource allocation.



## Dashboard Features

The Power BI dashboard includes the following features:

* Total number of government hospitals
* Total number of hospital beds
* State-wise distribution of hospitals
* State-wise distribution of hospital beds
* Region-wise comparison using pie and donut charts
* Hospitals vs Beds comparison chart
* Interactive slicers for filtering by Year and State
* Navigation buttons between report pages
* Reset filters functionality



## Dashboard Pages

### Page 1 – Hospitals & Beds Overview

This page provides a high-level summary of healthcare infrastructure including:

* Total Government Hospitals
* Total Hospital Beds
* Hospitals by State (Bar Chart)
* Beds by State (Column Chart)
* Filters for Year and State

### Page 2 – Hospitals & Beds Distribution

This page focuses on regional and comparative analysis:

* Hospitals by Region (Pie Chart)
* Beds by Region (Donut Chart)
* Hospitals vs Beds by State (Comparison Chart)
* Interactive filters and navigation

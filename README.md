# Government Hospitals and Beds Analysis

## Project Overview
This project analyzes government healthcare infrastructure in India using Power BI. 
The dashboard provides insights into the distribution of government hospitals and hospital beds across different states and regions.

## Dashboard Features
- Total number of government hospitals
- Total number of hospital beds
- State-wise hospital analysis
- State-wise bed distribution
- Region-wise distribution of hospitals and beds
- Interactive filters for Year and State
- Navigation between report pages

## Tools Used
- Power BI Desktop
- Microsoft Excel (Dataset preparation)
- GitHub for project hosting

## Dataset
Government Hospitals and Beds dataset containing information about:
- State
- Number of Government Hospitals
- Number of Hospital Beds
- Region type

## Report Pages

### Page 1 – Overview
- Total Hospitals
- Total Beds
- Hospitals by State
- Beds by State

### Page 2 – Distribution
- Hospitals by Region
- Beds by Region
- Hospitals vs Beds comparison
